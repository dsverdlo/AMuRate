Amurate

An Android Application where users can rate and look-up music and/or artists. Those scores are sent and maintained in a remote database.

Download .zip Download .apk View on GitHub

Bachelor project Computerscience VUB

This application is a Bachelor Project of the COMO Lab from the VUB (Vrije Universiteit Brussel). The promotor of this project is prof. Dr. Ann N with assistance of Maarten D. and Peter V.
